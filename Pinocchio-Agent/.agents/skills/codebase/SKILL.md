---
name: Codebase Search
description: Read the relevant files before editing and keep changes focused.
---

Start by inspecting the target files or nearby context with `Read` or `Bash`.

Before making changes:
- understand the existing pattern
- prefer the smallest safe edit
- avoid touching unrelated files

After making changes:
- verify the changed paths with `Read` or `Bash`
- summarize what changed and any follow-up risk
